6  Busca pelo Melhor Teste

Foi visto que os 3 testes apresentaram um desempenho semelhante, assim buscando encontrar o melhor teste, foi construida a seguinte tabela de comparação dos poderes de cada teste

6.1 Poder dos Testes

n_size_vector=c(500,1000,5000,10000)
alpha=5
alpha_vector=seq(4,6,by=0.2)
llhood_power=sapply(seq_along(n_size_vector),function(i){
  llhood_power_step=sapply(seq_along(alpha_vector),function(k){replicate(2000,rbeta(n=n_size_vector[i],shape1 = alpha_vector[k],shape2 = 1))%>%
    apply(2,llhood_ratio_test,alpha)%>%
    unlist()%>%
    sapply(power_decision)%>%
    sum()/2000}
)
  return(llhood_power_step)

})

6.2 Tempo de execução

llhood_time=sapply(seq_along(n_size_vector),function(i){
    summary(microbenchmark(
        replicate(5000,rbeta(n=n_size_vector[i],shape1 = 5,shape2 = 1))%>%
        apply(2,llhood_ratio_test,alpha)%>%
        unlist()%>%
        sapply(power_decision)%>%
        sum()/5000,
     times=100))[,4]%>%
     return()

})
wald_time=sapply(seq_along(n_size_vector),function(i){
    summary(microbenchmark(
        replicate(5000,rbeta(n=n_size_vector[i],shape1 = 5,shape2 = 1))%>%
        apply(2,wald_test,alpha)%>%
        unlist()%>%
        sapply(power_decision)%>%
        sum()/5000,
     times=100))[,4]%>%
     return()

})
score_time=sapply(seq_along(n_size_vector),function(i){
    summary(microbenchmark(
        replicate(5000,rbeta(n=n_size_vector[i],shape1 = 5,shape2 = 1))%>%
        apply(2,score_test,alpha)%>%
        unlist()%>%
        sapply(power_decision)%>%
        sum()/5000,
     times=100))[,4]%>%
     return()

})

Com isso, conclui-se que:

  • Um tamanho amostral de 500 amostras é o mínimo para um desempenho satisfatório nos 3 testes. É aconselhável um tamanho amostral \(\geq1000\) para pesquisas que necessitam de uma maior precisão

  • Testes Wald e Escore apresentaram convergencia a uma Distribuição \(X^2_1\) em tamanhos amostrais \(\geq10\) enquanto o Teste da Razão de Verossimilhança apresentou instabilidade para \(n\leq100\)

  • Poder dos 3 testes são semelhantes

  • Teste da Razão de Verossimilhança foi aquele que apresentou um maior tempo de execução, Teste Wald e Escore apresentaram tempos semelhantes.

Assim, para o modelo \(Beta(\alpha,1)\) os testes de Wald e Escore são os melhores a se utilizar
